# Starter template to write Node.js modules with TypeScript &middot; [![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/kevinpollet/typescript-node-module-starter/blob/master/LICENSE.md) [![Build Status](https://travis-ci.com/kevinpollet/typescript-node-module-starter.svg?branch=master)](https://travis-ci.com/kevinpollet/typescript-node-module-starter)

## Scripts

- `clean`: remove generated files and folders
- `lint:src`: run lint tool on source files
- `lint:test`: run lint tool on test files
- `build`: compile source files
- `format`: format all files
- `test`: run tests
